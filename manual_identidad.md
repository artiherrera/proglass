## 1. Concepto de marca

**ProGlass Care** desarrolla soluciones premium para **limpiar, proteger y conservar** artículos de uso diario. La marca convierte una rutina ordinaria de cuidado en una experiencia de calidad superior, con fórmulas inteligentes y resultados visibles.

**Promesa de marca:**

> **Cuidado premium para tu día a día.**

**Propuesta de valor:** una plataforma de productos de cuidado —calzado, lentes y cuidado personal— con estándar profesional, identidad consistente y orgullo de origen mexicano.

### Pilares de marca

| Pilar | Significado |
|---|---|
| **Calidad Premium** | Fórmulas y materiales de alta calidad para un cuidado superior. |
| **Confianza** | Transparencia y seguridad en cada producto que acompaña tu rutina. |
| **Innovación** | Soluciones inteligentes que marcan la diferencia. |
| **Resultados Reales** | Efectividad comprobada: una limpieza y protección que se nota. |
| **Hecho en México** | Orgullosamente desarrollado y producido en México, pensando en ti. |

## 2. Arquitectura de marca

ProGlass Care es una **marca paraguas** diseñada para crecer hacia múltiples categorías de cuidado. Cada sub-línea hereda el logotipo y el sistema visual, y se distingue por un **color de acento** propio.

```
                          PROGLASS CARE
                        (Cuidado Premium)
                                │
        ┌───────────────────────┼───────────────────────┐
 CUIDADO DE CALZADO        CUIDADO VISUAL          CUIDADO PERSONAL
        │                       │                       │
 ┌──────┼──────┬──────┐        │                       │
SNEAKERS GLOW FRESH SHIELD    VISION                   SOFT
```

| Línea | Producto | Color de acento | Descripción |
|---|---|---|---|
| Calzado | **SNEAKERS** | Azul `#0B5FE6` | Limpieza profunda para tenis y calzado. |
| Calzado | **GLOW** | Plata `#8C97A4` | Blanqueador profesional de suelas. |
| Calzado | **FRESH** | Verde `#34C759` | Eliminador de olores. |
| Calzado | **SHIELD** | Naranja `#FF7A1A` | Protección e impermeabilización. |
| Visual | **VISION** | Cian `#06B6D4` | Limpieza profesional para lentes. |
| Personal | **SOFT** | Violeta `#8B5CF6` | Espuma limpiadora para manos con colágeno. |

**Regla de oro:** el **azul ProGlass** siempre domina la identidad. Los colores de acento se usan solo para diferenciar categoría (etiquetas, íconos, fichas de colección), nunca para reemplazar el azul de marca.

## 3. Personalidad y tono de voz

**Atributos de personalidad:** premium · confiable · técnico pero cercano · resolutivo · mexicano.

**Cómo suena ProGlass:**

- **Claro y directo.** Frases cortas. Explica el beneficio antes que el ingrediente.
- **Seguro, no arrogante.** Respalda con resultados ("efectividad comprobada"), sin exagerar.
- **Cercano y profesional.** Habla de tú, en español de México, con cuidado en la gramática.
- **Orientado al resultado.** "Limpia, protege y conserva" — verbos de acción.

| Sí decimos | Evitamos |
|---|---|
| "Cuidado premium para tu día a día." | "El mejor producto del mundo." |
| "Resultados que se notan desde el primer uso." | Promesas absolutas o médicas no comprobadas. |
| "Hecho en México, pensado en ti." | Tecnicismos innecesarios o frialdad. |

**Palabras clave de marca:** premium, cuidado, protección, resultados, frescura, confianza, innovación.

## 4. Sistema de logotipo

El logotipo combina un **isotipo** (la "P" dinámica) con el **wordmark** "PROGLASS", el descriptor "CARE" y el lema "CUIDADO PREMIUM". Es un sistema flexible con versiones para cada aplicación.

### 4.1 Versión principal
![Logo principal a color](logos/png/proglass-principal.png)

Lockup vertical completo. **Uso:** comunicación institucional, digital e impresa de alto impacto.

### 4.2 Versión horizontal
![Logo horizontal](logos/png/proglass-horizontal.png)

**Uso recomendado:** sitios web, **encabezados (header)**, firmas de correo, presentaciones.

### 4.3 Versión vertical
![Logo vertical](logos/png/proglass-vertical.png)

**Uso recomendado:** packaging, etiquetas, material promocional.

### 4.4 Isotipo
![Isotipo](logos/png/proglass-isotipo.png)

La "P" como símbolo independiente. **Uso recomendado:** redes sociales, favicons, bordados, merchandising, ícono de app.

### 4.5 Versiones monocromáticas
![Mono negro](logos/png/proglass-mono-negro.png)

Monocromo negro (sobre fondos claros) y monocromo blanco (sobre fondos oscuros o de color). Para impresión a una tinta, grabados y aplicaciones donde el color no es viable.

### 4.6 Versiones sobre fondo
Cada lockup existe en dos tratamientos:

- **A color** — isotipo azul + cuerpo oscuro → sobre fondos **claros**.
- **`-dark`** — texto e isotipo en blanco (azul se conserva) → sobre fondos **oscuros o con fotografía**.

| Fondo | Versión a usar |
|---|---|
| Blanco / claro | A color (`proglass-*`) |
| Negro / Ink / foto oscura | Clara (`proglass-*-dark`) |
| Azul `#0B5FE6` | Monocromo blanco |
| Una sola tinta | Monocromo negro o blanco |

## 5. Construcción, área de seguridad y tamaño mínimo

**Área de seguridad.** Mantén alrededor del logotipo un margen libre **igual a la altura de la "P"** del isotipo. Ningún otro elemento (texto, imagen, borde) debe invadir esa zona.

```
   ┌───────────────────────────────┐
   │   ←x→                   ←x→    │   x = altura de la "P"
   │        [ LOGOTIPO ]           │
   │   ←x→                   ←x→    │
   └───────────────────────────────┘
```

**Tamaño mínimo** (para preservar legibilidad):

| Aplicación | Mínimo |
|---|---|
| Digital (versión horizontal) | **80 px** de ancho |
| Impreso (versión horizontal) | **20 mm** de ancho |
| Favicon / isotipo | 32 × 32 px |

## 6. Usos correctos e incorrectos

**✓ Hazlo**
- Conserva las proporciones originales y el contraste azul / oscuro.
- Usa la versión `-dark` o monocroma sobre fondos con foto o bajo contraste.
- Respeta el área de seguridad y el tamaño mínimo.

**✗ Evítalo**
- ❌ Deformar, estirar, comprimir o rotar el logotipo.
- ❌ Recolorear fuera de la paleta o eliminar la separación azul / oscuro.
- ❌ Aplicar sombras, contornos, degradados o efectos.
- ❌ Colocar el logotipo a color sobre fondos saturados o de bajo contraste.
- ❌ Reconstruir el wordmark con otra tipografía.
- ❌ Invadir el área de seguridad con otros elementos.

## 8. Tipografía

Dos familias gratuitas (Google Fonts), seleccionadas para reflejar el carácter técnico del logotipo y garantizar legibilidad en e-commerce.

### 8.1 Familia de títulos — **Saira**
Sans-serif técnica y cuadrada que hace eco del wordmark. Aporta fuerza y carácter premium.
**Pesos:** 600 SemiBold · 700 Bold · 800 ExtraBold (+ itálica para titulares de alto impacto).

### 8.2 Familia de texto — **Inter**
Sans-serif neutra y altamente legible, óptima para cuerpo, interfaz y datos.
**Pesos:** 400 Regular · 500 Medium · 600 SemiBold · 700 Bold.

### 8.3 Escala y jerarquía

| Nivel | Fuente | Tamaño | Interlínea / tracking |
|---|---|---|---|
| H1 / Hero | Saira 800 itálica · MAYÚSCULAS | 44–64 px | 1.05 · +1% |
| H2 / Sección | Saira 700 · MAYÚSCULAS | 30–40 px | 1.10 |
| H3 / Subtítulo | Saira 600 | 22–26 px | 1.20 |
| Cuerpo | Inter 400 | 16–18 px | 1.60 |
| Etiqueta / eyebrow | Inter 600 · MAYÚSCULAS | 12–13 px | +18% |
| Botón | Inter 600 | 14–16 px | +4% |

### 8.4 Alternativas
Si Saira/Inter no están disponibles (p. ej. en el selector nativo de un tema), usa:
**Títulos →** Archivo o Work Sans · **Texto →** Assistant o Work Sans.

## 9. Iconografía y elementos gráficos

**Estilo de íconos:** trazo lineal (line icons) de grosor uniforme, esquinas suaves, con un **destello / chispa azul** como acento recurrente (estrella de brillo de 4 puntas). Mantén los íconos monocromáticos (Ink o azul) salvo en la diferenciación por categoría.

**Elementos gráficos de apoyo:**
- **Glow azul:** halo de luz azul sobre fondos oscuros (firma visual del hero y banners).
- **Gotas y burbujas:** evocan limpieza, frescura y protección (impermeabilización).
- **Líneas de encuadre azules:** los pares de líneas que flanquean "CARE" pueden usarse como recurso para titulares y separadores.
- **Isotipo como patrón / marca de agua:** la "P" tenue, repetida o a gran escala, en paños, fondos y packaging.

---

## 10. Fotografía y estilo visual

**Dirección de arte:** producto premium sobre **fondos oscuros (Ink)** con **iluminación azul** lateral, gotas de agua y reflejos. Transmite tecnología, limpieza y cuidado de gama alta.

- **Sujetos:** tenis blancos, lentes, frascos de producto, paños de microfibra.
- **Iluminación:** clave baja con acento azul; contraste alto.
- **Atmósfera:** superficies húmedas, gotas, brillo; sensación de "recién cuidado".
- **Fondos claros:** alternativa limpia (blanco / Surface) para fichas de producto y catálogos, con sombras suaves.

### 12.2 Tipografía web

En `layout/theme.liquid`, dentro de `<head>`:

```html
<link rel="preconnect" href="https://fonts.googleapis.com">
<link rel="preconnect" href="https://fonts.gstatic.com" crossorigin>
<link href="https://fonts.googleapis.com/css2?family=Inter:wght@400;500;600;700&family=Saira:ital,wght@0,600;0,700;0,800;1,800&display=swap" rel="stylesheet">
```

En *Configuración del tema → CSS personalizado* (o `assets/custom.css`):

```css
h1, h2, h3, .h0, .h1, .h2, .banner__heading, .title {
  font-family: 'Saira', sans-serif !important;
  font-weight: 800; text-transform: uppercase; letter-spacing: .01em;
}
body, p, .product__text, button, .btn, input, select {
  font-family: 'Inter', sans-serif !important;
}
```
